# 🌦️ Weather App

This is a simple weather application built using OpenWeatherMap API.

## 🚀 Features
- Get real-time weather data
- Search weather by city name
- Displays temperature, humidity, and weather conditions

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
